Welcome to the API!

Options:

Get all tasks. - http://localhost:5000/api/telemetry/task?Page=
Get task with CallingAPIKey. - http://localhost:5000/api/telemetry/task?Calling_API_Key= &Page= &Page_Limit=
Post task - http://localhost:5000/api/telemetry/task

Calling_API_Key - String
Page - Integer > 0
Page_Limit - Integer between 1-100
